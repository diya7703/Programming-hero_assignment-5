Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select elements in the DOM. Here are some main methods:
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to get or set the HTML content of an element, which means it includes any HTML tags and renders them. On the other hand, innerText gets or sets only the visible text of an element, ignoring any text that is hidden with CSS, and it does not render HTML tags—instead, it shows them as plain text. Lastly, textContent gets or sets all the text content of an element, including hidden text, and like innerText, it treats HTML tags as plain text. It is also faster and more consistent than innerText because it doesn’t rely on styling.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where you add a single event listener to a parent element instead of each child element. When an event happens on a child, it bubbles up to the parent, and the parent can handle it using conditions. This makes your code more efficient, especially when dealing with many elements or dynamically added ones.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event starts from the target element and then moves up (or "bubbles") through its parent elements one by one. For example, if you click a button inside a div, the click event first triggers on the button, then on the div, and so on up to the root. This allows parent elements to react to events happening on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, you use document.createElement(). For example, const div = document.createElement('div') creates a new div element. To add it to the page, you can use methods like appendChild() or append() on a parent element, such as parent.appendChild(div). To remove an element, you can use removeChild() on its parent or simply call element.remove() if you have a reference to the element.
